## Quill Core

Quill Core is a document-first workflow kit.

- Treat these files as portable methodology assets.
- Do not turn Core into a runtime, model client, workflow engine, or memory system.
- Use the workflow, role-card, artifact-contract, review-gate, and memory docs together.

Core contents:

- `workflows/longform-writing.md`
- `skills/`
- `role-cards/technical-blog.md`
- `artifact-contracts/`
- `review-gates/`
- `memory/`
