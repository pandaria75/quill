# Skill: Longform Drafting

## Purpose

Produce a complete working draft without pretending it is already final.

## When To Use

- after the outline clears the draft gate
- when enough evidence exists to write through
- during major content generation

## Inputs

- intent
- angle
- structure
- context pack

## Outputs

- full draft
- visible placeholders for unresolved gaps

## Method

1. draft section by section from the structure
2. keep claims close to evidence
3. flag weak passages instead of smoothing them over
4. finish the full pass before micro-polish

## Quality Bar

- draft is complete end to end
- argument remains coherent
- unresolved issues are marked, not hidden

## Common Failure Modes

- polishing too early
- drifting away from the chosen angle
- inventing support for missing evidence
