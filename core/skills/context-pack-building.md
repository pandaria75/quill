# Skill: Context Pack Building

## Purpose

Assemble the minimum working context needed to draft safely and coherently.

## When To Use

- before outline or draft work
- when source material is scattered
- when handoff between people or agents is likely

## Inputs

- audience and intent
- chosen angle
- source notes and constraints

## Outputs

- compact context pack
- source list and key evidence notes
- explicit unknowns

## Method

1. collect only relevant sources
2. summarize constraints and risks
3. capture confirmed facts, assumptions, and unknowns
4. keep the pack short enough to scan quickly

## Quality Bar

- context is sufficient without being bloated
- evidence sources are traceable
- unknowns are explicit

## Common Failure Modes

- copying raw notes without synthesis
- hiding uncertainty
- adding irrelevant background
