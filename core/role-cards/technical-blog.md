# Role Card: Technical Blog

## Role

Technical-blog is a role-card for longform-writing work. It is not a workflow name.

## Responsibilities

- explain technical topics accurately
- keep claims traceable to sources or explicit reasoning
- balance clarity for the target audience with technical precision
- surface assumptions and open questions

## Deliverable Expectations

- strong problem framing
- clear structure
- verifiable claims
- revision-friendly Markdown output
